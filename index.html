FAQ

Common questions

Is this data live? +

Each module displays a status label so you can distinguish live, delayed, and demo values. Check the badge and source label on every card.

What sources do you use? +

Each card includes a source label and a last-updated timestamp. We pull from official feeds (BLS, Fed, ECB) and market APIs.

Does this replace Bloomberg? +

We position Planet News as a browser-based alternative for focused macro workflows, not a full Bloomberg terminal replacement.
